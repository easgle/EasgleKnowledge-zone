Dashboard Design with Power BI

สร้าง Impact ทางธุรกิจด้วย Dashboard ฉบับมือโปร

ถ้าคุณอยากจะขับเคลื่อนองค์กรของคุณ ให้เป็น Data-Driven Organization การสื่อสารกันด้วยข้อมูล
และการใช้ประโยชน์จากข้อมูลถือเป็นหนึ่งในปัจจัยสำคัญ ถ้าแม้จะมี Dashboard แต่ถ้าการออกแบบ หรือการนำเสนอ ไม่ตรงกับวัตถุประสงค์ของผู้ใช้งาน Dashboard นั้นก็ย่อมไม่ช่วยเพิ่ม/สร้างมูลค่าในกับธุรกิจ
การออกแบบ Dashboard ให้สามารถสร้าง Business Impact ได้จึงต้องอาศัยความเข้าใจทั้งเรื่องทฤษฎีการออกแบบ และการใช้งาน Tool ที่มีความยืดหยุ่นตามความต้องการของผู้ใช้ จัดเต็มกับ 3 หลักสูตร ให้คุณครบตั้งแต่ความเข้าใจด้านการออกแบบ Dashboard ที่ดี ไปจนถึงการใช้เครื่องมือเพื่อสร้างสิ่งที่คุณต้องการ
โดยเริ่มต้นตั้งแต่ระดับพื้นฐานไปจนถึงการใช้งาน Power BI ระดับสูง ผ่านการลงมือทำจริงทั้งแบบฝึกหัด และโปรเจกต์ ที่คุณสามารถนำไปประยุกต์กับธุรกิจของคุณได้หลังเรียนจบ

    แพ็กนี้เหมาะสำหรับ
  • ผู้ที่กำลังมองหาวิธีทำ Data Visualization/ Dashboard อย่างมีประสิทธิภาพ
  • Business Analyst หรือ Data Analyst ที่ต้องการวิเคราะห์ และสื่อสาร Insight ของข้อมูลให้เข้าใจง่ายขึ้น
  • ผู้ที่ต้องการเห็นภาพการใข้งานจริงของข้อมูลในเชิงธุรกิจ
  • ผู้ที่ต้องการเรียนรู้การใช้ Power BI ตั้งแต่พื้นฐานจนแอดวานซ์

      • หลังจากเรียนจบคอร์สนี้ผู้เรียนจะสามารถ
  • เข้าใจหลักวิธีการออกแบบ Dashboard ที่มีประสิทธิภาพ
  • เข้าใจหลัก และวิธีทำงานด้วย Power BI
  • เรียนรู้และสร้างความเข้าใจ สร้างความคุ้นชินกับข้อมูล ตลอดจนตัวอย่างการนำไปใช้
  • ทำ Data Visualization เพื่อนำไปประกอบการตัดสินใจ หรือเสนอบุคคลอื่นได้
  • เข้าใจ Frameworks ที่สำคัญๆ ทางธุรกิจ เช่น Pareto Analysis, Recency Frequency Monetary Analysis (RFM), Customer Lifecycle Stages Analysis
